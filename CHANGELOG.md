# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-02-03

### Added

- Initial public release
- Content-driven architecture with Nuxt 4 and @nuxt/content
- Markdown-based problem analysis articles with frontmatter support
- Status tracking system (Kutatás/Javaslat/Megvalósítás)
- Incremental solution steps component for structured problem breakdown
- Discussion links integration (LinkedIn, Bluesky)
- TL;DR summary blocks for articles
- Responsive design with Nuxt UI v4 and Tailwind CSS 4
- Dark mode support
- SEO optimization with sitemap generation and meta tags
- Accessibility enhancements with @nuxt/a11y
- Custom error page (404)
- TypeScript strict mode enabled

### Testing

- Unit tests with Vitest (14 tests)
- Component tests with Nuxt test environment (11 tests)
- E2E tests with Playwright (12 tests)
- CI workflow for automated testing on PRs

### Documentation

- Comprehensive README with setup instructions
